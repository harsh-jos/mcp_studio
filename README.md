# MCP Studio

MCP Studio is a Streamlit-based application that acts as a Model Context Protocol (MCP) client. It allows you to interact with MCP servers using a Gemini-powered AI agent.

## Getting Started

You can easily run this project using Docker Compose.

### How to Run

1.  **Start the application:**
    ```bash
    docker-compose up --build
    ```

2.  **Access the interface:**
    Open your browser and navigate to [http://localhost:8501](http://localhost:8501).

3.  **Configuration:**
    - Enter your Gemini API Key in the sidebar.
    - Configure your MCP servers using the JSON input in the sidebar.
